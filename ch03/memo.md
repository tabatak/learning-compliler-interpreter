### 数値定数の取得
文字コードから'0'を引いて数値を求める。
加算する前に10倍して桁をひとつ増やす。
```
char s[] = "1234";
inti, ch, num = 0;

for(i=0, ch=s[i]; isdigit(ch); ch=s[++i]){
  num = num*10 + (ch - '0');
}
```
